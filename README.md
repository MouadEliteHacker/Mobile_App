# Mobile_App
This project demonstrates a modern approach to movie discovery by combining traditional search functionality with AI-powered semantic recommendations. The application features a mobile-first interface built with React Native and an intelligent chatbot backend powered by a RAG (Retrieval Augmented Generation) system that provides context-aware movie recommendations.
